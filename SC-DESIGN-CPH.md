# SC-DESIGN-CPH | UI/UX Design System

## Layout Landing Page (Corporate Brochure)
1. **Hero Corporativo:** Imagen split-screen - izq: server room moderno / der: bodega de alimentos premium. Overlay oscuro con H1 "Soluciones Integrales para su Negocio"
2. **Trust Bar:** Certificaciones (FDA, USDA, ISO) + años de experiencia + clientes satisfechos
3. **Divisiones Grid:** Dos cards grandes lado a lado (50/50 en desktop)
   - Card Tech: Icono computadora, lista: "Hardware empresarial • Networking • Soporte técnico"
   - Card Food: Icono gourmet, lista: "Importación directa • Distribución nacional • Productos exclusivos"
4. **Proceso:** Timeline horizontal de 4 pasos (Importación → Control de Calidad → Logística → Entrega)
5. **Clientes:** Logos de empresas (placeholder) en grid monocromático
6. **Contacto Corporativo:** Formulario B2B serio + dirección + teléfono

## Componentes Específicos
- Nav: Clásica, logo centrado o izquierda, menú minimalista, botón "Contactar Ejecutivo"
- Cards división: Hover revela lista de servicios, border sutil dorado
- Tipografía: Playfair Display (títulos - elegancia) + Lato (cuerpo - legibilidad)
